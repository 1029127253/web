# web 此分支开箱即用

个人网站项目，前台包括博客、代码库、文件下载、留言、登录注册等功能，后台包括上传文件、博客、代码，编辑、删除文章，查看、修改个人资料及邮箱、头像等功能。

PC端网址：www.hemingsheng.cn ，移动端网址：https://m.hemingsheng.cn

## 主要文件结构如下

- `docs` -> 说明文档
- `web-core` -> 后端Java源代码
- `web-mobile` -> 响应式界面源码（在web和移动端是响应式布局）
- `web-pc` -> pc端界面源码（非响应式布局）
